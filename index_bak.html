Merge Tags

Merge Tags allow you to copy a code snippet that will be replaced with the dynamic data when the ecard template is sent.

For example, if your template includes "[sender_name] sent you an ecard" and the sender's name is "Tim", then when the the e-ecard is sent the text will say "Tim sent you an ecard".

Below are the Merge Tags available for you to use in the Subject, Heading, and/or Body of your e-card template:
Name Merge Tag Code
Personal Message Text
Using this merge tag will disable the standard personal message block. 		[personal_message]
Formatted Personal Message Box
Using this merge tag will allow you to place the formatted personal message box anywhere in the ecard body. 		[personal_message_box]
Sender's name [senders_name]
Sender's email address [senders_email]
Recipient's name [recipients_name]
Recipient's email address [recipients_email]
